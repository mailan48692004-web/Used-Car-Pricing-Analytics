# Used Car Pricing Analytics: Price Driver Analysis and Market Pattern Exploration
Built a pricing analytics project using Python to analyse large-scale used car listing data and identify the factors most strongly associated with vehicle prices. The project involved data cleaning, feature engineering, exploratory analysis, and price-driver investigation across vehicle attributes such as age, mileage, condition, transmission, fuel type, and location.

## Project Overview
This project was developed as a pricing analytics case study using a large real-world used car listings dataset. The goal was to understand how different vehicle and listing characteristics were associated with price variation in the used car market. Because the dataset was large, messy, and commercially realistic, the project required substantial preprocessing, variable selection, and feature engineering before meaningful analysis could be conducted. The project was designed to support business-style pricing insight and market understanding, rather than to produce a final production-ready valuation model.

## What I Did
Cleaned a large used car listings dataset and addressed data quality issues such as missing values, inconsistent categories, and noisy listing fields
Selected and transformed key pricing-related variables including price, age, mileage, condition, fuel type, transmission, and location
Engineered features to improve analytical usability and interpretability
Explored price distribution and market patterns across major vehicle segments
Analysed the relationship between price and core attributes to identify the most important price drivers
Generated business insights on how vehicle characteristics and market context influence observed listing prices

## Tools
Python, Pandas, NumPy, Data Cleaning, Feature Engineering, Exploratory Data Analysis, Pricing Analytics, Descriptive Statistics, Data Visualisation, Large Dataset Handling, Business Insight Generation

## Business Value
The project demonstrates how messy marketplace data can be turned into structured pricing insight. By identifying the main attributes associated with higher and lower listing prices, the analysis can support pricing decisions, market scanning, and value comparison across used vehicle segments. It also shows the practical challenge of working with large, imperfect commercial data rather than idealised classroom datasets.

## Note
This project is an exploratory pricing analytics case study based on observed used car listings. The analysis identifies associations and market patterns, not causal price effects. Listing prices may also differ from final transaction prices, so the results should be interpreted as market-facing pricing insight rather than validated valuation outcomes.
